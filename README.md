# Titanic Data Cleaning & Visualization Project

## Objective
The objective of this project is to clean, process, analyze, and visualize the Titanic dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Data Cleaning Performed
- Handled missing values in the Age column using mean imputation.
- Filled missing values in the Embarked column using mode.
- Removed the Cabin column due to excessive missing values.
- Checked and handled duplicate records.

## Visualizations
- Gender Distribution
- Survival Distribution
- Survival by Gender
- Survival by Passenger Class
- Age Distribution

## Key Insights
1. Female passengers had a higher survival rate than male passengers.
2. First-class passengers had the highest survival rate.
3. Most passengers belonged to the 20–40 age group.
4. More passengers died than survived.
5. Male passengers outnumbered female passengers.

## Conclusion
This project demonstrates the complete workflow of data cleaning, preprocessing, visualization, and extracting meaningful insights from raw data.
